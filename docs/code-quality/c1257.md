# <a name="c1257"></a>C1257
Warnung 1257: Fehler beim Initialisieren des Plug-ins.

 Das Code Analysetool meldet diese Warnung, wenn ein interner Fehler im Plug-in vorliegt, nicht im analysierten Code.
