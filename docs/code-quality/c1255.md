# <a name="c1255"></a>C1255
Warnung C1255: PCH-Daten für das Plug-in weisen eine falsche Länge auf.

 Das Code Analysetool meldet diese Warnung, wenn ein interner Fehler im Tool vorliegt, nicht im analysierten Code.
