## <a name="example"></a>Beispiel
 Der folgende Code zeigt Code, der Warnung C26166 generiert.

```cpp

typedef struct _DATA {
CRITICAL_SECTION cs;
} DATA;

_Acquires_lock_(p->cs) void Enter(DATA* p) {
```
