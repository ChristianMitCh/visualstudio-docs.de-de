# <a name="c28206"></a>C28206
Warnung C28206: \<Ausdrucks >: Linker Operand zeigt auf eine Struktur, verwenden Sie `->`

 Diese Warnung wird gemeldet, wenn die `->` der Struktur Zeiger dereferenzierungsnotation erwartet wurde.
