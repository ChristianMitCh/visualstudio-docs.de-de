# <a name="list-of-available-services"></a>Liste der verfügbaren Dienste

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]und das Visual Studio SDK unterstützen die folgenden Dienste. Einige Pakete bieten ihre eigenen Dienste an, die hier nicht aufgeführt sind, z. B. haben Sprachdienste keine einzige Dienst-GUID. Sie müssen den Namen der Sprache verwenden, um die GUID des Sprachdienstes in der Registrierung zu finden.

Verwenden Sie die hier aufgeführten oder von einer anderen Quelle abgerufenen Dienst-GUIDs (z. B. Sprachdienste), um die primäre Schnittstelle oder die mit jedem Dienst angezeigten Schnittstellen abzusondern.
