# <a name="idebugengine2enumprograms"></a>IDebugEngine2::EnumPrograms
Ruft eine Liste aller Programme ab, die von einem Debugmodul (DE) gedebuggt werden.

## <a name="syntax"></a>Syntax

```cpp
HRESULT EnumPrograms( 
   IEnumDebugPrograms2** ppEnum
);
```

```csharp
int EnumPrograms( 
   out IEnumDebugPrograms2 ppEnum
);
```

## <a name="parameters"></a>Parameter
`ppEnum`\
[out] Gibt ein [IEnumDebugPrograms2-Objekt](../../../extensibility/debugger/reference/ienumdebugprograms2.md) zurück, das eine Liste aller Programme enthält, die von einem DE gedebuggt werden.

## <a name="return-value"></a>Rückgabewert
 Wenn die Ausführung erfolgreich ist, wird `S_OK`, andernfalls ein Fehlercode zurückgegeben.
