# <a name="idebugdisassemblystream2getscope"></a>IDebugDisassemblyStream2::GetScope
Ruft den Bereich des Demontagestreams ab.

## <a name="syntax"></a>Syntax

```cpp
HRESULT GetScope( 
   DISASSEMBLY_STREAM_SCOPE* pdwScope
);
```

```csharp
int GetScope( 
   out enum_ DISASSEMBLY_STREAM_SCOPE pdwScope
);
```

## <a name="parameters"></a>Parameter
`pdwScope`\
[out] Gibt einen Wert [DISASSEMBLY_STREAM_SCOPE](../../../extensibility/debugger/reference/disassembly-stream-scope.md) aus der DISASSEMBLY_STREAM_SCOPE-Enumeration zurück, die den Umfang dieses Demontagestroms beschreibt.

## <a name="return-value"></a>Rückgabewert
 Wenn die Ausführung erfolgreich ist, wird `S_OK`, andernfalls ein Fehlercode zurückgegeben.

## <a name="remarks"></a>Bemerkungen
 Der Umfang einer Demontage kann z.B. eine Funktion oder das gesamte Modul sein.
