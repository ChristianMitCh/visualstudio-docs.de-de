# <a name="ieedatastoragegetsize"></a>IEEDataStorage::GetSize
Gibt die Anzahl der in diesem Objekt enthaltenen Bytes zurück.

## <a name="syntax"></a>Syntax

```cpp
HRESULT GetSize(
   ULONG* size
);
```

```csharp
int GetSize(
   out uint size
);
```

## <a name="parameters"></a>Parameter
`size`\
[out] Die Anzahl der in diesem Objekt enthaltenen Bytes.

## <a name="return-value"></a>Rückgabewert
 Wenn die Ausführung erfolgreich ist, wird `S_OK`, andernfalls ein Fehlercode zurückgegeben.

## <a name="remarks"></a>Bemerkungen
 Verwenden Sie die [GetData-Methode,](../../../extensibility/debugger/reference/ieedatastorage-getdata.md) um die tatsächlichen Datenbytes abzurufen.
