# <a name="idebugarrayfieldgetelementtype"></a>IDebugArrayField::GetElementType
Ruft den Elementtyp im Array ab.

## <a name="syntax"></a>Syntax

```cpp
HRESULT GetElementType( 
   IDebugField** ppType
);
```

```csharp
int GetElementType(
   out IDebugField ppType
);
```

## <a name="parameters"></a>Parameter
`ppType`\
[out] Gibt ein [IDebugField-Objekt](../../../extensibility/debugger/reference/idebugfield.md) zurück, das den Elementtyp beschreibt.

## <a name="return-value"></a>Rückgabewert
 Wenn erfolgreich, kehrt S_OK zurück; Andernfalls wird ein Fehlercode zurückgegeben.

## <a name="remarks"></a>Bemerkungen
 Das [IDebugArrayField-Objekt](../../../extensibility/debugger/reference/idebugarrayfield.md) geht davon aus, dass alle Elemente des Arrays vom gleichen Typ sind.
